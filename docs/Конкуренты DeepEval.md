##  Прямые конкуренты DeepEval

### Ragas (RAG Assessment)

**Ragas** — open-source библиотека, специализирующаяся исключительно на RAG-системах.

| Аспект                  | Описание                                                                                                              |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------- |
| **Что делает**          | Оценивает каждый компонент RAG: ретривер (precision/recall контекста), генератор (faithfulness, релевантность ответа) |
| **Отличие от DeepEval** | Более глубокая аналитика по RAG (метрики для ретривера), но хуже для обычных чат-моделей                              |
| **Плюсы**               | Лучшие метрики для RAG, интеграция с синтетическими тестами, активное сообщество                                      |
| **Минусы**              | Бесполезен вне RAG, сложнее в настройке кастомных судей                                                               |

### Phoenix (Arize)

**Phoenix** — платформа для observability и оценки LLM (open-source версия бесплатна, есть облачный SaaS).

| Аспект                  | Описание                                                                                        |
| ----------------------- | ----------------------------------------------------------------------------------------------- |
| **Что делает**          | Трейсинг запросов, визуализация эмбеддингов (проекция в 3D), детекция аномалий, оценка качества |
| **Отличие от DeepEval** | Это не просто тесты, а полноценный мониторинг в продакшене с дашбордами                         |
| **Плюсы**               | Бесплатная open-source версия, красивые графики для анализа ошибок, поддержка агентов           |
| **Минусы**              | Тяжеловесный, требует запуска сервера, сложнее для простых unit-тестов                          |

### LangSmith (LangChain)

**LangSmith** — коммерческая платформа от создателей LangChain (есть бесплатный тариф с ограничениями).

| Аспект                  | Описание                                                                                   |
| ----------------------- | ------------------------------------------------------------------------------------------ |
| **Что делает**          | Всё в одном: трейсинг, отладка, тестирование, датасеты, мониторинг продакшена              |
| **Отличие от DeepEval** | Интегрирован с LangChain на уровне экосистемы, но закрытый код и привязка к вендору        |
| **Плюсы**               | Мощнейший инструмент для агентов и цепочек, отличный UI, активная поддержка                |
| **Минусы**              | Платная модель (после 5000 трассировок), vendor lock-in, не для локального использования   |

### Promptfoo

**Promptfoo** — инструмент командной строки для стресс-тестирования промптов.

| Аспект                  | Описание                                                                              |
| ----------------------- | ------------------------------------------------------------------------------------- |
| **Что делает**          | Сравнивает промпты и модели между собой по заданным метрикам                          |
| **Отличие от DeepEval** | Конфигурация через YAML, акцент на A/B тестирование промптов, а не на логику кода     |
| **Плюсы**               | Очень быстрый, отличный для сравнения моделей (GPT vs Llama), минималистичный         |
| **Минусы**              | Нет сложных метрик (faithfulness, контекстная релевантность), всё нужно писать самому |

### LoCoBench — это не фреймворк, а бенчмарк

LoCoBench - это комплексный тест, специально разработанный для оценки моделей большого языка с длительным контекстом (LLM) в сложных сценариях разработки программного обеспечения. Он предоставляет 8000 сценариев оценки на 10 языках программирования с длиной контекста от 10 тыс. до 1 млн токенов.

### Важное отличие от DeepEval

|DeepEval|LoCoBench|
|---|---|
|Инструмент, который вы **запускаете для своей модели**|Экзамен, который **проверяет модели**|
|Можно кастомизировать под свои нужды|Фиксированный набор задач (8,000 тестов)|
|Используется в CI/CD для контроля качества|Используется в исследованиях для сравнения моделей|

### Что измеряет LoCoBench?

- **Длина контекста**: от 10k до 1 млн токенов
- **Задачи**: понимание архитектуры, поиск багов, рефакторинг, анализ безопасности, мульти-язычность (10 языков)
- **Метрики**: 17 штук, включая "архитектурную согласованность" (оценивается LLM-судьёй)
