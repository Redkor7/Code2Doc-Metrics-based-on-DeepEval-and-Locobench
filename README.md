# Code2Doc-Metrics-based-on-DeepEval-and-Locobench

Данный проект направлен на изучение различных метрик для измерения качества работы LLM (в частности метрик в deepEval и Locobench), а также на написание
своих метрик с помощью DeepEval. Основное направление - это метрики для оценки LLM, которые пишут документацию по исходному репозиторию проекта.

## В данный момент написаны следующие метрики: 
- code_text_coherence - метрика, которая определяет насколько связаны между собой блок кода в документации и его описание текстом.
- semantic_completeness - метрика, которая проверяет описала ли нейросеть все важные аспекты проекта в документации или что-то было упущено. Использует "золотой стандарт" (документацию, которая считаеся хорошо соответствующей проекту, написанная или проверенная людьми) при сравнении.

## Инструкция как запустить тесты в DeepEval:

1. Создать виртуальное окружение Python (если не было создано)
`python -m venv venv`
2. Запустить его
`venv\Scripts\activate`
3. Установить deepEval (если не устновлен)
`install -U deepeval`
4. Установить OPENAI_API_KEY (если не установлен) 
`set OPENAI_API_KEY=...``
5. Установить базовый URL при необходимости(использование proxy) 
`OPENAI_BASE_URL=https://api.proxyapi.ru/openai/v1`
6. написать тест с препиской в файле `test_...`
7. Запустить тест deepeval 
`deepeval test run test_e.py`
